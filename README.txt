This project looks into the influence of rewards on selective attention using Steady state visual evoked potentials. This directory includes preprocessed data as well as preprocessing and analysis scripts.

Authors: Ivan Grahek*, Antonio Schettino*, Gilles Pourtois, Ernst Koster, & Søren Andersen
(*: co-first authors)

The codes for analyses were written by Ivan Grahek and Antonio Schettino during 2016-2018.
