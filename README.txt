This project investigates the influence of reward on selective attention using steady-state visual evoked potentials. This repository includes preprocessed data as well as preprocessing and analysis scripts.

Authors: Ivan Grahek*, Antonio Schettino*, Ernst Koster, & Søren Andersen
(*: co-first authors)

The analysis scripts were written by Ivan Grahek and Antonio Schettino during 2016-2019.

Contents:

1. Behavioral preprocessing
	- Matlab script for the preprocessing of the behavioral data

2. EEG_preprocessing 
	- Matlab scripts for preprocessing of the EEG data

3. analysis
	- R scripts for fitting the models and summarizing the results

4. data
	- Pre-processed behavioral and EEG data

5. figures
	- Figures for the manuscript

6. manuscript
	- The manuscript
