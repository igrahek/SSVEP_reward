interp_elecs {eegUtils}R Documentation

Channel interpolation

Description

Interpolate EEG channels using a spherical spline (Perrin et al., 1989). The data must have channel locations attached.

Usage

interp_elecs(data, bad_elecs, ...)

## S3 method for class 'eeg_data'

Arguments

data

Data as a eeg_data or eeg_epochs object.

bad_elecs

Name(s) of electrode(s) to interpolate.
