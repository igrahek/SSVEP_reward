compute_psd {eegUtils}R Documentation

Compute power spectral density

Description

Returns PSD calculated using Welch's method for every channel in the data. Currently concatenates across epochs. Returns uV^2 / Hz

Usage

compute_psd(data, ...)

## S3 method for class 'eeg_data'
compute_psd(data, seg_length = NULL, noverlap = 0,
  n_fft = 256, srate = NULL, method = "Welch")

Arguments

data

Data to be plotted. Accepts objects of class eeg_data

...

any further parameters passed to specific methods

seg_length

Length of rolling data segments.

noverlap

Number of (sampling) points of overlap between segments.

n_fft

Length of FFT to be calculated.

srate

Sampling rate

method

Defaults to "Welch". No other method currently implemented.
